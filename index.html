Architecture

Generate → Validate → Execute

A deterministic checkpoint inserted between AI generation and runtime. Every command. Every time. No exceptions.

STEP 01
Code Generated

AI agent, developer, or CI pipeline generates a command — shell script, SQL, CI/CD YAML, or npm manifest.

STEP 02
Sent to Sentinel

Command routed to Sentinel via Telegram bot or API gateway. Zero config. No SDK changes required.

STEP 03
Risk Engine Analyzes

25 named rules scan for wiper patterns, eval/exec abuse, injection vectors, secret exposure, malicious packages, and unsafe CI permissions.

STEP 04
Verdict Returned

Structured decision returned with rule ID, risk score 0–100, and human-readable explanation.

🛑 BLOCK ⚠️ WARN ✅ ALLOW
STEP 05
Execution Decision

Execution allowed or stopped. Blocked threats cost zero credits. Agent receives structured output to act on immediately.

STEP 06
Audit Logged

Every decision recorded. Full audit trail for enterprise compliance, government reporting, and sovereign deployments.
