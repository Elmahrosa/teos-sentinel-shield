# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability within TEOS Sentinel Shield, please email  
**security@elmahrosa.dev**. We will respond within 72 hours and provide a timeline for resolution.

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| main    | :white_check_mark: |

## Dependencies

We use Dependabot to monitor and patch known vulnerabilities in our supply chain.  
Critical patches are merged within 24 hours of availability.
